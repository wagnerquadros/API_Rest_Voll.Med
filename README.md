# API Rest Voll.Med

API REST utilizando Spring Boot desenvolvida na formação "Java e String Boot" da Alura.

## Estrutura do Projeto

#### Configurações:
- `maven/dependencies`:
  - Spring Web
  - Spring Boot DevTolls
  - Longbok
  - Spring Data JPA
  - Flayway Migration
  - MySQL Driver
  - Validation


- `Como executar`: executar a classe `src/main/java/med.voll.api/ApiApplication`.
